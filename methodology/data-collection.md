# 数据收集方法

## 数据来源

### 一手来源
- GitHub 源码仓库
- 官方文档
- 官方博客

### 二手来源
- 技术社区讨论（Hacker News、Reddit、V2EX）
- 技术博客文章
- 学术论文

## 收集工具

| 工具 | 用途 |
|------|------|
| Git | 源码克隆与版本管理 |
| IDE | 源码阅读与分析 |
| 浏览器 | 文档与社区信息收集 |

## 数据记录

- 所有引用的资料都记录在 [references/](../references/) 目录
- 重要发现及时记录在对应的分析文档中
- 保持数据的时效性，标注收集日期
