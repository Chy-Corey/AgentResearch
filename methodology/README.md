# 研究方法论

## 研究目标

横向对比 Claude Code、Codex、Hermes、OpenClaw 四个 AI Agent，从源码角度深入分析其架构设计、技术实现和工程实践。

## 研究流程

```
1. 信息收集 → 2. 源码分析 → 3. 横向对比 → 4. 总结输出
```

### 阶段一：信息收集
- 收集各 Agent 的官方文档、GitHub 仓库、技术博客
- 了解产品定位、功能特性、技术栈

### 阶段二：源码分析
- 阅读项目结构和入口文件
- 分析核心模块和关键算法
- 梳理依赖关系和设计模式

### 阶段三：横向对比
- 按维度逐项对比
- 制作对比矩阵
- 提炼关键差异

### 阶段四：总结输出
- 撰写综合评价
- 给出适用场景建议
- 形成研究结论
